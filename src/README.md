## 体验使用 WebGAL 制作的游戏

[铃色☆记忆](http://hoshinasuzu.cn/) by 星奈组；在这里可以观看，[该游戏的发布视频](https://www.bilibili.com/video/BV1HY4y1n7z7)

[Elf of Era Idols Project](https://store.steampowered.com/app/2414730/Elf_of_Era_Idols_Project/) (通过 Steam 获取)
